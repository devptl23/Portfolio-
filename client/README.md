## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/devptl23/web-developmemt.git
cd portfolio/client
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.

## ğŸ¨ Customization

### Update Personal Information

1. **Contact Details**: Edit `src/contact.jsx`
2. **Projects**: Modify the projects array in `src/projects.jsx`
3. **Experience**: Update the timeline in `src/components/Home.jsx`

- Global styles: `src/index.css`
- Component-specific styles: `src/styles/*.css`
- Color scheme: Update CSS custom properties in the root styles
